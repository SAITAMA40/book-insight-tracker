# Book Insight Tracker

A web application for tracking insights and quotes from books.

## Features

- Add and manage books
- Add insights and quotes for each book
- Review and edit insights
- Tag system for better organization
- Responsive design

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Radix UI Components

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Deployment

The application is deployed on Vercel. Visit [https://book-insight-tracker.vercel.app](https://book-insight-tracker.vercel.app) to view the live version.
